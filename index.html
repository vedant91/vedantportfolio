What I'm Building

The CYPHEX Roadmap

Three pillars that turn CYPHEX from a scanner into a living cyber defense system.

01
🔌

CYPHEX IoT — Edge Security Appliance

A $200 plug-and-play hardware device (Raspberry Pi 5) that runs the entire 11-agent pipeline locally on your network. Zero cloud dependency. The AI brain (Llama 3.2 3B) runs on-device via Ollama — no code ever leaves your premises. Physical LED indicators show real-time security status. Continuous scan cycles every 1–6 hours with automated alerting.

Raspberry Pi 5OllamaEdge AIZero Cloud
02
🧬

Self-Learning Vulnerability Model

An AI model that doesn't just scan — it learns. CYPHEX builds a per-endpoint "Behavioral Genome" of your application using Isolation Forest ML. It profiles what normal looks like (input patterns, response sizes, request rates) and blocks anything abnormal — even AI-generated zero-day attacks it has never seen before. Daily CVE feed ingestion keeps the model updated with new vulnerability patterns automatically.

Isolation ForestAnomaly DetectionCVE Auto-PatchBehavioral ML
03
🛡️

Future Goal — Application Immune System

The endgame: CYPHEX becomes a full Application Immune System. A reverse proxy shield that sits in front of any web app, scoring every request in real-time (<1ms). When a vulnerability is confirmed, AI auto-generates a code patch and opens a GitHub PR — while the shield protects the app until the developer reviews. Every CYPHEX instance shares anonymized threat intelligence, creating herd immunity across all users.
